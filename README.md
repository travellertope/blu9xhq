# Bluu HQ

## Repository Structure

```
bluuhq/
├── theme/          # Custom WordPress theme (bluuhq.com)
└── portal/         # Headless CRM & client portal (portal.bluuhq.com)
```

### `theme/`
Custom WordPress theme powering the agency website at bluuhq.com.

### `portal/`
Headless WordPress-connected CRM and client portal deployed at portal.bluuhq.com.
Built as a decoupled frontend that consumes the WordPress REST API / WPGraphQL.
